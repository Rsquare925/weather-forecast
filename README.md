# Weather Forecast App

A simple weather forecast application built using HTML, Tailwind CSS, and JavaScript.

## Table of Contents

-   [Features](#features)
-   [Installation](#installation)
-   [Usage](#usage)
-   [Technologies Used](#technologies-used)

## Features

-   Get current weather information for any location.
-   Get 5 day weather forecast for any location or current location.
-   Displays temperature, humidity, wind speed, and weather conditions.
-   User-friendly interface designed with Tailwind CSS.
-   Responsive design for mobile and desktop views.

## Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/Rsquare925/weather-forecast.git
    cd weather-forecast
    ```

2. **Open the `/src/index.html` file in your preferred browser:**

    ```bash
    open src/index.html
    ```

    Alternatively, you can use a local server to serve the files.

## Usage

1. **Search for a city:**

    - Enter the name of a city in the search bar.
    - Press the "Search" button or hit "Enter".

2. **View weather information:**

    - The app will display the current weather data for the entered city.
    - Information includes temperature, weather conditions, humidity, and wind speed.

3. **5 day forecast for searched city or current location:**
    - The app will display the next 5 day weather data for the entered city or current location.

## Technologies Used

-   **HTML**: The standard markup language used for creating web pages.
-   **Tailwind CSS**: A utility-first CSS framework for rapidly building custom user interfaces.
-   **JavaScript**: A programming language that enables interactive web pages and is an essential part of web applications.
-   **OpenWeatherMap API**: A weather API used to fetch current weather data for various locations.
